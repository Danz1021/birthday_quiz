# 🎮 團康賓果猜題遊戲

一個專為團體活動設計的互動式猜題賓果遊戲，主持人操作、現場作答，連線獲勝！

---

## 🧠 這是什麼？

這是一個跑在瀏覽器上的**團康遊戲工具**，不需要安裝任何軟體，直接開啟 `index.html` 即可使用。

遊戲採用**賓果（Bingo）機制**：
- 題板共有 **5×5 = 25 格**，每格代表一道題目
- 答對 → 格子顯示 ⭕
- 答錯 → 格子顯示 ❌
- 任意橫列、直行、對角線湊成 **五個 ⭕ 連線** → 賓果！獲勝 🎉

---

## 🎯 支援的題型

| 題型 | 說明 |
|---|---|
| 💬 文字問答 | 主持人念題，現場口頭回答 |
| 🎵 音樂猜題 | 顯示歌詞片段，猜歌名／歌手 |
| 🖼️ 圖片猜謎 | 用表情符號＋文字提示描述對象，猜答案 |
| 📋 選擇題 | 四個選項，讓現場搶答或投票 |

---

## 🕹️ 操作流程

```
開始遊戲
   ↓
點選任意格子（選題）
   ↓
主持人念題 / 現場作答
   ↓
主持人點「揭示答案」→ 確認答案
   ↓
點「答對了 ⭕」或「答錯了 ❌」判定
   ↓
回到題板，格子自動更新
   ↓
五連線 → 賓果！進入勝利畫面 🎉
```
