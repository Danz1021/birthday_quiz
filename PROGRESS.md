# 生日賓果猜題 — 工作進度紀錄

## 專案概述
為酥酥、師父、劉宇謙三位壽星設計的生日派對互動賓果遊戲。  
主辦方操作，三位壽星作答，連成三條線獲勝。

---

## 檔案結構

```
bingo-quiz/
├── index.html       ← 主遊戲（目前工作版 v2）
├── index.v1.html    ← v1 備份（Claude 暖棕主題）
├── index.v2.html    ← v2 備份（炫彩紫色主題，含計時器/音效/翻牌）
├── 生日題庫2.pdf    ← 原始題目來源
├── PROGRESS.md      ← 本文件
└── README.md        ← 專案說明
```
